# DecodeLabs Internship - project 1
## Java Number Guessing Game


#Project Description

This is a simple Java console-based Number Guessing Game.

The program generates a random number between **1 and 100**, and the player has to guess it. After each guess, the program provides feedback:

-  If the guessed number is too low, it displays **"Your guess number is low."**
-  If the guessed number is too high, it displays **"Your guess number is high."**
-  If the guess is correct, it displays **"You guessed the right number!"** and ends the game.

The game continues until the player guesses the correct number.

---

## Features

- Random number generation (1–100)
- User input using `Scanner`
- Feedback for high and low guesses
- Repeats until the correct number is guessed
- Exception handling for invalid input
